# VirtualMeet Icons

This directory contains the application icons for VirtualMeet.

## Icon Sizes Required

- **32x32.png** - Small icon
- **128x128.png** - Medium icon
- **128x128@2x.png** - Large icon (256x256)
- **icon.icns** - macOS icon
- **icon.ico** - Windows icon

## Design Guidelines

The VirtualMeet icon should represent:
- Virtual presence/communication
- Video and media capabilities
- Professional and modern appearance
- Blue/purple color scheme for tech feel
